**Title:** Raspberry Pi Image Description Provisioning

|                           |             |
| ------------------------- | ----------- |
| **Type**                  | `object`    |
| **Required**              | No          |
| **Additional properties** | Not allowed |

**Description:** The Image Description Provisioning (IDP) document describes a Raspberry Pi image for secure on-device provisioning. It combines image layout information (partition table, partition images, sizes, filesystem types and attributes) with a Provisioning Map (PMAP) that defines how the image is deployed — including partition creation, LUKS encryption, and security attributes. IDP documents are consumed by Raspberry Pi provisioning tools such as rpi-sb-provisioner.

| Property                     | Pattern | Type   | Deprecated | Definition | Title/Description                                                                                                                                                                                                              |
| ---------------------------- | ------- | ------ | ---------- | ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| + [IGversion](#IGversion )   | No      | string | No         | -          | The IDP document format version in semver format (MAJOR.MINOR.PATCH). Provisioning tools use this field to select the correct parsing and provisioning path, enabling backward compatibility across document format revisions. |
| + [IGmeta](#IGmeta )         | No      | object | No         | -          | Build configuration metadata derived from the rpi-image-gen environment. For images built outside rpi-image-gen these fields must be populated manually.                                                                       |
| + [attributes](#attributes ) | No      | object | No         | -          | Image-level attributes describing the overall image geometry and identity.                                                                                                                                                     |
| + [layout](#layout )         | No      | object | No         | -          | The image layout, comprising the partition table, partition images, and provisioning map.                                                                                                                                      |
