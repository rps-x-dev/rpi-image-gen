Raspberry Pi Image Description Provisioning

Type: object

The Image Description Provisioning (IDP) document describes a Raspberry Pi image for secure on-device provisioning. It combines image layout information (partition table, partition images, sizes, filesystem types and attributes) with a Provisioning Map (PMAP) that defines how the image is deployed — including partition creation, LUKS encryption, and security attributes. IDP documents are consumed by Raspberry Pi provisioning tools such as rpi-sb-provisioner.
